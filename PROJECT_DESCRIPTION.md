# Project Description: NPIP

Nepal Public Figure Intelligence Platform (NPIP) is a research-focused intelligence product that
collects public information about Nepali public figures, verifies sources, and presents the latest
activities in a concise timeline. The MVP targets journalists, researchers, NGOs, and political
analysts who need fast, trustworthy context.

## MVP Scope

- Identity layer: Wikipedia search + summary
- Data aggregation: GNews search for verified headlines
- Output: profile card, activity timeline, recent news

## Next Milestones

1. Disambiguation and multi-entity confirmation
2. Additional sources (YouTube, X/Twitter, Nepali portals)
3. Deduplication + sentiment pipeline
4. Alerts, digests, and comparison dashboards
